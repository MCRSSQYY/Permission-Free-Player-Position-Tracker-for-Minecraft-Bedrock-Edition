# Permission-Free-Player-Position-Tracker-for-Minecraft-Bedrock-Edition
#### 一款用于为Minecraft BE没有OP权限的玩家用来定位其他玩家坐标的工具
***
# 原理
## 指令原理
Minecraft BE中的私信指令"/w"与目标选择器"@a\[ \]"的配合
#### /w
用于告诉自己@s
#### @a\[ \]
用于选择符合\[ \]中条件的所有玩家
###### \[ \]
\[ \]中填写x,y,z,dx,dy,dz选中其标记的区域
## 生成原理
输入
利用二分法逐渐缩小玩家坐标的范围
输出
